# 📊 Data Analysis Dashboard

A Flask web app that analyzes student marks and displays visual charts.

## 🛠️ Tech Stack
- Python, Pandas, Matplotlib, Flask

## ✨ Features
- Load CSV data automatically
- Shows Average, Highest, Lowest marks
- Displays Bar chart visualization

## 🚀 How to Run
```bash
pip install flask pandas matplotlib
python app.py
```
Open browser → `http://localhost:5000`
